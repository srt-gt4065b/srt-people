# 🚀 SRT 회원 DB - 초간단 시작 가이드

## Windows 사용자

### 방법 1: 배치파일 사용 (초간단!)

1. **실행하기.bat** 파일을 프로젝트 폴더에 복사
2. **실행하기.bat** 더블클릭
3. **[1]** 입력하고 Enter (처음 설치)
4. 설치 완료되면 다시 **실행하기.bat** 더블클릭
5. **[2]** 입력하고 Enter (개발 서버 실행)
6. 브라우저에서 **http://localhost:5173** 접속
7. 비밀번호 **20251214** 입력

### 방법 2: 명령어로 직접 실행

```cmd
# 1. 프로젝트 폴더로 이동 (파일 탐색기 주소창에 'cmd' 입력)

# 2. 패키지 설치 (처음 한 번만)
npm install

# 3. 개발 서버 실행
npm run dev
```

## Mac/Linux 사용자

### 방법 1: 쉘 스크립트 사용

```bash
# 1. 실행하기.sh를 프로젝트 폴더에 복사

# 2. 실행 권한 부여 (처음 한 번만)
chmod +x 실행하기.sh

# 3. 스크립트 실행
./실행하기.sh

# 4. [1] 입력 (처음 설치)
# 5. 다시 ./실행하기.sh 실행
# 6. [2] 입력 (개발 서버 실행)
```

### 방법 2: 명령어로 직접 실행

```bash
# 1. 프로젝트 폴더로 이동
cd ~/경로/srt-people-main

# 2. 패키지 설치 (처음 한 번만)
npm install

# 3. 개발 서버 실행
npm run dev
```

## ❌ 문제 해결

### Node.js가 없다고 나옴
→ https://nodejs.org 에서 LTS 버전 설치

### npm이 명령어로 인식 안 됨
→ 컴퓨터 재시작

### 포트가 이미 사용 중
→ 다른 개발 서버 종료 또는:
```bash
npm run dev -- --port 3000
```

### 설치 중 오류
```bash
npm cache clean --force
npm install
```
